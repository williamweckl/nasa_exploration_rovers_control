    NASAExplorationRoversControl.move_exploration_rover

    You're seeing just the function move_exploration_rover, go back to NASAExplorationRoversControl module for more information.
    Link to this function

    move_exploration_rover(exploration_rover)

    View Source

    Moves an exploration rover according to its current direction. The result of this function will be the exploration rover with a new position.

    Examples 

    iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "N"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,1}, direction: "N"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,3}, direction: "N"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,4}, direction: "N"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,3}, direction: "S"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,2}, direction: "S"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,1}, direction: "S"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "S"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {2,0}, direction: "W"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {1,0}, direction: "W"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {1,0}, direction: "W"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "W"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {3,0}, direction: "E"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {4,0}, direction: "E"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "E"})
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {1,0}, direction: "E"}}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "S"})
{:error, "Invalid position. Coordinates must not be negative."}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "W"})
{:error, "Invalid position. Coordinates must not be negative."}

iex> move_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "I"})
