    API Reference NASA Exploration Rovers Control v1.0.0

    Modules

    NASAExplorationRoversControl

    Bounded context for the NASA Exploration Rovers Control System. This module contains all the system use cases.

    NASAExplorationRoversControl.CelestialBodies.Factory

    Factory responsible to interact with inputs and return the Celestial Body module accordingly.

    NASAExplorationRoversControl.CelestialBodies.Mars

    Module representing Mars Celestial Body.

    NASAExplorationRoversControl.ExplorationRover

    Exploration Rover entity, it represents a real exploration rover that is exploring another planet.

    NASAExplorationRoversControl.Interactors.ExecuteExplorationRoverCommands

    Use case responsible for execute an exploration rover given commands.

    NASAExplorationRoversControl.Interactors.ExploreCelestialBodyUsingCommandsFromFile

    Use case responsible for use the commands instructions from file to interact with the exploration rovers to explore a Celestial Body.

    NASAExplorationRoversControl.Interactors.InterpretExplorationCommandsInputFile

    Use case responsible for interpret exploration commands input file and return a readable map.

    NASAExplorationRoversControl.Interactors.MoveExplorationRover

    Use case responsible for move an exploration rover.

    NASAExplorationRoversControl.Interactors.RotateExplorationRover

    Use case responsible for rotate an exploration rover.

    NASAExplorationRoversControlTerminalInterface

    This module is a bridge between the user and the application business logic (aka. Delivery Mechanism).

    NASAExplorationRoversControlTerminalInterface.CelestialBodiesMenu

    This module contains logic to show terminal interface celestial bodies menu to user and to handle user interactions.

    NASAExplorationRoversControlTerminalInterface.CommandsExecutor

    This module contains logic to execute commands from selected file by user.

    NASAExplorationRoversControlTerminalInterface.CommandsInputFilesMenu

    This module contains logic to show terminal interface command input files menu to user and to handle user interactions.

    NASAExplorationRoversControlTerminalInterface.FinalStep

    This module contains for the final step at the interaction with the terminal interface.

    NASAExplorationRoversControlTerminalInterface.IO

    This module contains helpers for interacting with the user's terminal.

    Mix Tasks

    mix start_terminal_interface

    Automated task to start the terminal interface that is a way that users can interact with the system business logic and have the chace to make a difference to the world by exploring another celestial bodies.
