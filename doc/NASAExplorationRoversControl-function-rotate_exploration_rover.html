    NASAExplorationRoversControl.rotate_exploration_rover

    You're seeing just the function rotate_exploration_rover, go back to NASAExplorationRoversControl module for more information.
    Link to this function

    rotate_exploration_rover(exploration_rover, direction)

    View Source

    Rotates an exploration rover. The result of this function will be the exploration rover with a new direction.

    Examples 

    iex> rotate_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "N"}, "L")
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "W"}}

iex> rotate_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "S"}, "L")
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "E"}}

iex> rotate_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "W"}, "L")
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "S"}}

iex> rotate_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "E"}, "L")
{:ok, %Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "N"}}

iex> rotate_exploration_rover(%Elixir.NASAExplorationRoversControl.ExplorationRover{position: {0,0}, direction: "I"}, "L")
